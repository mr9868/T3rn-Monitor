## T3rn executor Monitor server

installation step :
```
apt install bc jq curl;
git clone https://github.com/mr9868/T3rn-Monitor;
cd T3rn-Monitor;
nano config;
chmod +x t3rn.sh;
./t3rn.sh
```
> [!IMPORTANT]
> This is only works on linux system !
> You must fill required variable on the config file !
